
AI Resume Analyzer & Job Matcher

Description:
This project analyzes uploaded PDF resumes and matches skills against predefined job roles.
It uses Flask for backend APIs and Streamlit for a modern interactive dashboard.

Features:
- PDF resume parsing
- Skill extraction
- Job-role matching
- REST API backend
- Clean Streamlit UI dashboard

Tech Stack:
Python, Flask, Streamlit, PyPDF2

Run Instructions:

1. Install dependencies:
pip install -r requirements.txt

2. Start backend:
python app.py

3. Launch dashboard:
streamlit run dashboard.py

Then upload resume PDF and view job match analytics.
